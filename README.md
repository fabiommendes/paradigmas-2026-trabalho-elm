# Lista de exercícios de Elm

Neste exercício simples, implementaremos algumas operações com árvores binárias. A avaliação é feita usando testes unitários.

## Instruções

* Clone este repositório via Github Classroom
* Instale o elm e o elm-test usando pacotes da sua distribuição ou sudo `npm
  install -g elm elm-test`.
* Execute `elm-test` para rodar os testes.
* Implemente as funções de cada módulo para que os testes passem.
* Faça commit e push das suas alterações para o repositório remoto.
* **O trabalho é individual.** Não é permitido compartilhar código, mas é
  permitido discutir as soluções com outros alunos.

## Critérios de avaliação e correção

* Os testes devem passar na suite de testes original e não devem ser
  modificados.
* Posso trocar valores dos testes na correção final para verificar se a
  implementação é correta ou está direcionada apenas para passar em testes
  específicos.
* Cada teste que passar a partir do oitavo vale 5%. Por exemplo, se
  passaram 20 testes no total, a nota será (20 - 8) * 5% = 60%.
* Este trabalho corresponde à 5% da nota total.
